# End Effector Prototypes Repository

This repository contains the code for the first and second end effector prototypes, as described in the research chapters.

---

 | [0005_Schadensfall_2_Greifen von Strut und befestigen an Joint](0001_First%20end%20effector%20prototype/0005_Schadensfall_2_Greifen%20von%20Strut%20und%20befestigen%20an%20Joint) | Code for damage case 2: gripping the strut and attaching it to the joint. |
